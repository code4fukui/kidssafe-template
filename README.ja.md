# キッズセーフ 〇〇市 〇〇地区

〇〇市〇〇地区の地域安全マップを作成し、スマホやパソコンで共有することができるツールです。ExcelやNumbersを使ってCSVデータを編集してアップロードすることで、マップの更新が簡単にできます。[キッズセーフ by Code for FUKUI](https://github.com/code4fukui/kidssafe/)を使って作られています。

## デモ
https://code4fukui.github.io/kidssafe-template/

## 機能
- ExcelやNumbersでCSVデータを編集・アップロードすることで、地域安全マップを更新できる
- データの種類やアイコンを追加することができる

## 使い方
1. 変更したいデータのCSVファイルをダウンロードし、Excelで編集する
2. 編集したファイルを[./](./)にドロップしてアップロードする
3. 1分ほど待つと更新される

詳しい手順は[README.md](README.md)を参照してください。
